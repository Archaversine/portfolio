Evolutionary Computation - Smart Rockets

Posted on April 3, 2023 by Adam Brohl

A remake of this project, which was based off of TheCodingTrainâ€™s p5js project.

In this project, small rockets try to reach the red circle on the other side of the wall. If they hit the wall, then they die and cannot progress any further. The rockets eventually get better and better at going around the wall via a genetic algorithm.

Genetic Algorithm Parameters

The following parameters are used for the genetic algorithm:

Parameter Value
Population Size 200
Lifespan (Frames) 500
Crossover Rate 80%
Crossover Type Single Point
Mutation Rate 1%
Elitism Top 1
